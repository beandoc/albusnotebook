Unofficial Library

Empower your research with NotebookLM

The ultimate Python API and CLI for programmatic access to Google's NotebookLM. Automate sources, chat history, and rich content generation.

pip install notebooklm-py

Use at Your Own Risk

This library uses undocumented Google APIs that can change without notice. It is a community project and not affiliated with Google.

What You Can Build

AI Agent Tools

Integrate NotebookLM into Claude Code or other LLM agents. Ships with Claude Code skills for natural language automation.

Research Automation

Bulk-import sources (URLs, PDFs, YouTube, Google Drive) and run web/Drive research queries with auto-import.

Content Generation

Generate Audio Overviews, videos, slide decks, quizzes, flashcards, infographics, and mind maps with full control.

Downloads & Export

Batch download artifacts locally (MP3, MP4, PDF, PNG, CSV, JSON, Markdown) or export to Google Docs/Sheets.

Diverse Content Formats

Go beyond simple text. Generate immersive Audio Overviews, whiteboard-style videos, interactive mind maps, and structured study aids. notebooklm-py gives you full programmatic control over styles, lengths, and output formats.

Simple to Use

import asyncio
from notebooklm import NotebookLMClient

async def main():
    async with await NotebookLMClient.from_storage() as client:
        # Create notebook and add sources
        nb = await client.notebooks.create("Research")
        await client.sources.add_url(nb.id, "https://example.com", wait=True)

        # Chat with your sources
        result = await client.chat.ask(nb.id, "Summarize this")
        print(result.answer)

        # Generate content
        status = await client.artifacts.generate_audio(nb.id)
        await client.artifacts.download_audio(nb.id, "podcast.mp3")

asyncio.run(main())
# Authenticate (opens browser)
notebooklm login

# Create and use notebook
notebooklm create "My Research"
notebooklm use <notebook_id>

# Add sources and chat
notebooklm source add "./paper.pdf"
notebooklm ask "What are the key themes?"

# Generate and download
notebooklm generate audio "--wait"
notebooklm download audio "./podcast.mp3"
# Install via CLI
notebooklm skill install

# Ask Claude to research
"Create a podcast about quantum computing"
"Download the quiz as markdown"
"/notebooklm generate video"
